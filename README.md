# Glossary for Craft CMS

![Icon](resources/glossary.png)

## Background

A glossary with tooltips.

## Requirements

 * Craft CMS >= 3.5.0

## Installation

Open your terminal and go to your Craft project:

``` shell
cd /path/to/project
composer require codemonauts/craft-glossary
./craft install/plugin glossary
```
